# Interior Point Method Convergence Improvement - Thesis

This repository contains the code and other files used to work on my thesis, which is an attempt at improving the convergence of the interior point method.

The interior point method is a numerical optimisation method used for quadratic problems with linear constraints.

The essential idea of this thesis is that when we start getting closer to the central path, we try and tackle the convergence issues, especially due to the lack of invertibililty of the system of equations as we get closer to a local solution.
